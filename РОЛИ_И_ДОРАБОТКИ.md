## 2. Разграничение доступа по ролям

### Сборщик (sborshik1-4)
- Видит: Главная (только свои сборки), Сборка, Детали (только просмотр)
- НЕ видит: Приход, Заявки, Отчёты, Настройки, Журнал операций, Ячейки
- Боковое меню: Главная, Сборка, Детали

### Кладовщик (kladovshik)
- Видит: Главная (дашборд остатков), Остатки, Приход, Расход, Ячейки, Заявки, Журнал операций
- НЕ видит: Отчёты, Настройки пользователей
- Боковое меню: всё кроме Отчётов и Настроек

### Мастер (master)
- Видит всё кроме управления пользователями
- Дополнительно: Отчёты с аналитикой

### Администратор (admin)
- Видит всё включая управление пользователями и настройки системы

## 4. Новые страницы

### /details — Справочник деталей
- Таблица всех 53 деталей
- Колонки: Артикул, Наименование, Двигатель, Ед.изм., Месячный план, Ячейка, Текущий остаток, Статус (светофор)
- Поиск по артикулу и названию
- Фильтр по типу двигателя (ТВ2-117 / ТВ3-117)
- Для мастера и админа — кнопка редактировать пороги

### /reports — Отчёты (мастер и выше)
Три вкладки:
1. **Расход за период** — график по дням, фильтр по детали и двигателю
2. **Сборки** — таблица всех двигателей: номер, сборщик, дата начала/окончания, статус
3. **Дефицит** — список позиций которые были в красной зоне за период

### /cells — Ячейки хранения
- Таблица всех ячеек: код, стеллаж, полка, деталь которая там лежит, количество
- Визуальная схема стеллажа (простая сетка где видно занятые/свободные ячейки)
- Поиск ячейки по коду

### /settings — Настройки (только admin)
Вкладки:
1. **Пользователи** — список, кнопка добавить/редактировать/удалить
2. **Пороги светофора** — таблица деталей с полями красный/жёлтый порог, можно редактировать
3. **Тема** — переключатель тёмная/светлая тема (сохраняется в localStorage)
4. **Операции сборки** — список операций по типам двигателей, можно добавлять

- Предупреждения о критических остатках

## 6. Технические детали

### Сессии и авторизация
- При входе сохранять в сессию: user_id, username, role, full_name
- Декоратор `@login_required` на все маршруты
- Декоратор `@role_required('master', 'admin')` для защищённых маршрутов
- При попытке зайти без прав — редирект на /login с сообщением

### Переключатель темы
- Кнопка в боковом меню снизу (рядом с именем пользователя)
- Иконка 🌙/☀️
- Сохранять выбор в localStorage
- CSS переменные уже заданы — добавить класс `light-theme` на body

### Logout
- Кнопка выхода в боковом меню снизу
- После выхода — редирект на /login
